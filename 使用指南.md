## 🛠️ 常用操作

### 启动服务
```bash
# 后端
cd backend
python -m uvicorn app.main:app --reload --port 8001

# 前端  
cd frontend
npm run dev
```

### 停止服务
在对应的终端窗口按 `Ctrl+C`

### 查看日志
两个服务都会在终端中显示实时日志

## 💡 使用技巧

1. **首次使用**：建议先注册账户，然后创建一些测试待办事项
2. **团队协作**：可以打开两个浏览器窗口，用不同账户登录测试实时同步
3. **离线测试**：断开网络连接，操作应用后再恢复网络观察同步效果
4. **WebSocket测试**：使用专门的测试页面可以更好地理解实时同步机制

## 🆘 常见问题

**Q: 页面无法访问？**
A: 检查终端中服务是否正常运行，确认端口没有被占用

**Q: 登录失败？**
A: 确保后端服务正在运行，检查网络连接

**Q: 实时同步不工作？**
A: 检查WebSocket连接状态，在开发者工具中查看网络面板

**Q: 数据丢失？**
A: 应用有本地存储保护，刷新页面通常能恢复数据
